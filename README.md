### init




```
node-project
├── package.json
├── index.js 
├── app
|   ├── router.js
|   ├── error_handler.js (default error handler)
│   ├── controller
│   |   └── home.js
│   ├── middleware (optional)
│   |   └── response_time.js
│   ├── init (optional)
│   |   └── init.js
│   ├── config (optional)
│   |   └── config_dev or config_prod
│   ├── model (optional)
│   |   └── xxx.js
```
